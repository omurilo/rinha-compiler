# rinha-compiler

Interpretador em Go feito para [rinha de compiler](https://github.com/aripiprazole/rinha-de-compiler).

## Features

- [x] Int, Str, Bool
- [x] Binary
- [x] Let
- [x] Function
- [x] If
- [x] Call
- [x] Print
- [ ] First
- [ ] Second
- [ ] Tuple

## Build

```bash
docker build -t rinha .
```

## Run

```bash
docker run -it rinha examples/fib-tc.rinha
```
