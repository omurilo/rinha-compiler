# rinha-compiler

Interpretador em Go feito para [rinha de compiler](https://github.com/aripiprazole/rinha-de-compiler).

## Features

- [x] Int, Str, Bool
- [x] Binary
- [x] Let
- [x] Function
- [x] If
- [x] Call
- [x] Print
- [x] First
- [x] Second
- [x] Tuple


## Build

```bash
make build
```

## Run

```bash
make hello
make showcase
make bench
make test file=examples/sum.json
```
